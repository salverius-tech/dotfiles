# typescript-testing

TypeScript/JavaScript testing practices with Bun's built-in test runner, covering test structure, assertions, mocking, async patterns, coverage configuration, and project organization.

## Files

| File | Description |
|------|-------------|
| `knowledge.md` | Full testing guidelines and patterns for Bun projects |

## Topics

- Bun test execution (never jest/vitest in Bun projects)
- Test file organization and discovery patterns
- Assertions, setup/teardown, and lifecycle hooks
- Mocking with `mock()`, module mocking, and spies
- Async/await and Promise testing
- Coverage configuration via `bunfig.toml`
- Integration testing (HTTP APIs, databases)
- React component testing with Bun
