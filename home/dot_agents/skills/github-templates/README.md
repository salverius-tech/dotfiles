# github-templates

Guidelines for GitHub repository configuration, community files, issue/PR templates, CODEOWNERS, branch protection, and Copilot instructions.

## Files

| File | Description |
|------|-------------|
| `knowledge.md` | Full skill guidelines and reference |
| `assets/CODEOWNERS.template` | Code ownership pattern template |
| `assets/CONTRIBUTING.md.template` | Contribution guidelines template |
| `assets/SECURITY.md.template` | Vulnerability reporting process template |
| `assets/bug_report.yml.template` | Bug report issue form template |
| `assets/copilot-instructions.md.template` | Copilot workspace context template |
| `assets/pull_request_template.md.template` | Pull request template |

## Topics

- Essential repository files (README, LICENSE, CONTRIBUTING, SECURITY)
- Issue templates with YAML form format
- Pull request templates and review workflows
- CODEOWNERS configuration and review assignments
- Branch protection and GitHub Actions CI
- Dependabot and .gitattributes configuration
- GitHub Copilot workspace instructions
