# React Workflow Skill

Guidelines for React projects covering component architecture, React 19 features, Server Components, testing with Testing Library, accessibility, state management, and composition patterns.

## Files

| File           | Purpose                                                |
| -------------- | ------------------------------------------------------ |
| `knowledge.md` | Portable React workflow guidelines (loaded by agents)   |

## Topics Covered

- Functional component architecture with TypeScript props interfaces
- React 19 features (useActionState, useFormStatus, useOptimistic, use())
- Server Components (RSC) and client/server boundary patterns
- Testing with Vitest + Testing Library (accessible queries, user-event, async)
- Accessibility (ARIA, keyboard navigation, focus management)
- State management hierarchy (local, lifted, context, external store)
- Custom hooks, composition patterns, error boundaries, and Suspense
