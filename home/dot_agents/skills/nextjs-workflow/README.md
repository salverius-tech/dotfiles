# Next.js Workflow Skill

Guidelines for Next.js projects covering App Router, Server/Client Components, file-based routing, data fetching, caching, image/font optimization, middleware, and TypeScript configuration.

## Files

| File           | Purpose                                                  |
| -------------- | -------------------------------------------------------- |
| `knowledge.md` | Portable Next.js workflow guidelines (loaded by agents)  |

## Topics Covered

- App Router structure, route groups, parallel and intercepting routes
- Server Components vs Client Components (`'use client'` directive)
- Server Actions for form handling and data mutations
- Data fetching with request deduplication, caching, and on-demand revalidation
- Image optimization (`next/image`) and font optimization (`next/font`)
- Metadata API (static, dynamic, template patterns)
- Middleware for auth, headers, and rewrites
- Environment variables (`NEXT_PUBLIC_` prefix convention)
