# Docs Workflow Skill

Documentation workflow guidelines covering Markdown standards, specification types (RFCs, ADRs, design docs), Mermaid diagrams, and CI validation.

## Files

| File           | Purpose                                                  |
| -------------- | -------------------------------------------------------- |
| `knowledge.md` | Portable documentation workflow guidelines (loaded by agents) |

## Topics Covered

- Markdown standards (headings, lists, code blocks, line length)
- Mermaid diagram best practices
- Specification types (RFCs, ADRs, design docs)
- YAML frontmatter metadata
- Cross-references and linking (internal, external, anchors)
- CI validation with markdownlint and link checking
- README structure and file organization
