# Django Workflow Skill

Django framework workflow guidelines covering architecture patterns, model organization, settings, views, forms, admin, migrations, testing, and security.

## Files

| File           | Purpose                                              |
| -------------- | ---------------------------------------------------- |
| `knowledge.md` | Portable Django workflow guidelines (loaded by agents) |

## Topics Covered

- Tool grid (Ruff, mypy, django-stubs, bandit, pytest-django, squawk)
- Django 5.x features (composite PKs, async views, CSP, background tasks)
- Architecture patterns (services pattern, selectors pattern)
- Model organization and field ordering conventions
- Settings organization with environment-specific packages
- URL patterns, views (CBV vs FBV), forms, and admin customization
- Migration guidelines (schema vs data migrations, non-nullable fields)
- Testing with pytest-django and security checklist
