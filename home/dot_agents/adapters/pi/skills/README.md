# Pi Skill Wrappers

Place Pi-specific Agent Skill wrappers here when a canonical `~/.agents/skills/<name>/knowledge.md` file needs Pi-specific activation metadata or tool instructions.

The global Pi settings currently also load `~/.agents/adapters/opencode/skills`, which already contains `SKILL.md` files compatible with Pi's Agent Skills discovery. Add wrappers here for Pi-only overrides.
