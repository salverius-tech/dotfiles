Create or update a project status snapshot at `$1` if provided, otherwise use `STATUS.md`.

Include:
- Current branch and high-level goal
- Completed work
- Current state
- Next actions
- Known risks/blockers
- Commands/tests recently run and their results

Keep it concise and useful for resuming a future agent session.
