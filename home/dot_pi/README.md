# Pi Configuration

Managed global configuration for [Pi coding agent](https://pi.dev).

Installed target layout:

```text
~/.pi/agent/
├── AGENTS.md
├── settings.json
├── models.json
├── keybindings.json
├── prompts/
└── extensions/
```

Runtime data such as sessions, auth files, and installed packages must remain local and should not be committed.
